# ContentAPI
API for adding custom content to PaperMC-based servers

## Features
- Registry system
- Argument types for `com.mojang.brigadier`
  - Custom item
- PersistentData types
  - `org.bukkit.NamespacedKey`
- Custom items
- Custom recipes (WIP)
  - Crafting

See [wiki](https://github.com/Kiber2009/ContentAPI/wiki) for more information
